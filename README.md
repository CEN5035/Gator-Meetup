# GatorMeetup
Website to organize events within UF
Tech Stack : Angular, Angular Material, Node.js, Express, MongoDB

## Install/Run instructions  
##### Frontend
    npm install;
    ng serve -o
##### Backend
    npm install;
    node server.js
